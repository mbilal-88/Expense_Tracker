# Expense Tracker (Python)

A simple **command-line Expense Tracker** built with Python. It records expenses, saves them to a CSV file, and shows a summary including total spending, category-wise expenses, remaining budget, and daily budget.

## Features

* Add expenses with category selection
* Save expenses to a **CSV file**
* Category-wise expense summary
* Remaining monthly budget and daily spending limit

## Run with uv

```bash
uv run expense_tracker.py
```

## Files

* `expense_tracker.py` – Main program
* `expense.py` – Expense class
* `expenses.csv` – Stores expense data
